# Miui Camera For Fleur & Miel
## Ported from MIUI Camera Module by [itzdfplayer](https://t.me/itzdfplayer)
## How to use?
   1. Clone this repo to ```packages/apps/MIUICamera```
   2. Add this line ```PRODUCT_PACKAGES += MIUICamera MiuiExtraPhoto privapp-permissions-miuicamera.xml miuicamera-hiddenapi-package-whitelist.xml sceneDetection.xml``` to your device.mk
